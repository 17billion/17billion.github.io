#### 시간복잡도(Time complexity) 표기법

알고리즘의 시간복잡도를 표기할때의 규칙
Big-O(빅-오), Big-Ω(빅-오메가), θ(세타) 표기

```
Big-O : 최악의 경우
Big-Ω: 최선의 경우
θ> Big-O, Big-Ω의 평균
```

보통 표기는  Big-O를 사용, 최악의 경우을 고려하는게 필요하기 때문
